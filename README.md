# Frubana Frontend - test

prueba para desarrollador frontend de frubana.

### Requisitos
* nodejs
* yarn o npm

### iniciar el proyecto
* clonar el repositorio:

  ``` bash
  $ git clone https://github.com/Frubana/bodega_cachubana.git
  ```
* instalar dependencias
  ``` bash
  $ cd carpeta-del-proyecto
  $ yarn install
  ```
* iniciar el servidor
  ``` bash
  $ yarn start
  ```
